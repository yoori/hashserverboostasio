Сервер генерирующий murmurv3 hash для поступающих по tcp соединениям строк (разделенных \n)
Реализован на основе boost.asio - мой первый опыт использования данной библиотеки,
в принципе производительность не плохая - system cpu usage примерно в полтора раза больше user cpu usage
на восьмиядерной машине в режиме 5 потоков получается ~700K rps (размер строк = 100 байт)

Компиляция (стандартная при использовании cmake):
mkdir build
cd build
cmake ..
make

Запуск:
./bin/BAHashServer -p <порт> -t <кол-во потоков>
